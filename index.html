Project Bug & Hacking

DDOS

DDoS stands for Distributed Denial of Service, which is a cyberattack that aims to make a website or online service inaccessible. This attack is carried out by flooding a website with unauthorized traffic.

Bug 404

404 bug bounty is a bug bounty related to 404 errors, which are error codes that appear when a user tries to access a non-existent web page.

Bug hunter Parameter

The parameters of a bug bounty are the rules that govern what can be tested, how to report findings, and how much the reward will be.

Evillimiter

illegally controlling the flow of network traffic that can harm network users.

Sniifing (MITM) Bettercap

Sniffing is a cyber crime committed by intercepting personal data over the internet. The stolen data can include usernames, passwords, credit card information, and other important data.

Sqlmap

SQL Injection attacks involve the insertion or “setup” of SQL queries via input data from the client to the application. A successful attack allows the attacker to manipulate the SQL queries that the application makes to its database.

Osint Maltego

Maltego is an OSINT application that provides a platform to not only extract data but also present the data in a format that is easy to understand and analyze.

CSRF

CSRF (Cross-Site Request Forgery) is an attack that forces users to perform actions without their consent. This attack takes advantage of browser security policies that allow automated requests without user consent.

WPscan

WPScan is a free and open-source WordPress security scanning tool. WPScan is used to identify vulnerabilities in WordPress sites, such as vulnerabilities in plugins, themes, and WordPress core.

RapidScan

RapidScan can refer to a ticket scanning application, document scanning application, or web vulnerability scanning tool.

Xerosploit (MITM)

MITM stands for Man-in-the-Middle, which is a cyberattack carried out by intercepting communication between two parties. This attack aims to steal the victim's data or credentials.

Reverse Shell

A shell is a computer program that interacts with the operating system such as the Windows terminal and Bash. Common operations with the shell include login/logout, printing process information, accessing/deleting files, and connecting to remote systems.

Campish

CamPhish is a phishing tool that allows attackers to create fake websites that ask victims to provide personal details that give access to the front camera of the victim's smartphone.

Mr Holmes

Mr.Holmes is an information gathering tool (OSINT). Its main purpose is to obtain information about domains, usernames, and phone numbers with the help of public sources available on the internet, also using google dorks attacks for certain researchers. This tool also uses proxies to make your request completely anonymous and WhoIS Api to obtain more information about a domain.
